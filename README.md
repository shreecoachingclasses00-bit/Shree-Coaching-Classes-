# Shree-Coaching-Classes-
Shree Coaching Classes is a 6-year-old local tutoring center  in Lakhanpuri, Kanker (Chhattisgarh), offering KG to 12th  grade coaching  with specialized  preparation for Navodaya Vidyalaya, Prayas, and Eklavya exams.Only coaching center in the area  offering FREE WiFi and laptop access for Class 8th-12th  students to support digital learning.
